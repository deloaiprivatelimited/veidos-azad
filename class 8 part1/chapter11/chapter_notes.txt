## 1. Chapter Overview

### Core theme
- ಭೂಮಿಯ ಮೇಲ್ಮೈಯ ಪ್ರಮುಖ **ಭೌತಿಕ ಭೂಸ್ವರೂಪಗಳು**:
  - **ಪರ್ವತಗಳು, ಪ್ರಸ್ಥಭೂಮಿಗಳು, ಮೈದಾನಗಳು, ಮರುಭೂಮಿಗಳು, ದ್ವೀಪಗಳು**
- ಇವುಗಳ:
  - **ವ್ಯಾಖ್ಯಾನ**, **ಉಗಮ/ರಚನೆ**, **ವರ್ಗೀಕರಣ**, **ಉದಾಹರಣೆಗಳು**, **ಮಾನವ ಜೀವನದ ಮೇಲೆ ಪ್ರಭಾವ**
- ಜಾಗತಿಕ ಮಟ್ಟದಲ್ಲಿ **ಹವಾಮಾನ + ಭೌಗೋಳಿಕ ಲಕ್ಷಣಗಳ ಆಧಾರದಲ್ಲಿ** ರೂಪುಗೊಂಡ **ಸ್ವಾಭಾವಿಕ ಪ್ರದೇಶಗಳು (Natural Regions)**

### Why important for GPST / HSTR
- **ಭೌತಿಕ ಭೂಗೋಳಶಾಸ್ತ್ರದ ಮೂಲಾಧಾರ**: endogenic–exogenic ಶಕ್ತಿಗಳ ಪಾತ್ರ, ಭೂಸ್ವರೂಪ–ಮಾನವ ಚಟುವಟಿಕೆ ಸಂಬಂಧ.
- ಪರೀಕ್ಷೆಯಲ್ಲಿ ಸಾಮಾನ್ಯವಾಗಿ ಕೇಳುವವು:
  - **ವರ್ಗೀಕರಣ + ಉದಾಹರಣೆಗಳು** (1–2 ಅಂಕ)
  - **ಆರ್ಥಿಕ/ಸಾಮಾಜಿಕ ಪ್ರಾಮುಖ್ಯತೆ** (5 ಅಂಕ)
  - **ನಕ್ಷೆ/ಚಿತ್ರ ಆಧಾರಿತ ಗುರುತింపు** (landforms + natural regions)
- ಕರ್ನಾಟಕ ಸಂಬಂಧಿತ ಅಂಶ:
  - **ದಖನ್ ಪ್ರಸ್ಥಭೂಮಿ (Karnataka ಭಾಗ)**, **St. Mary’s Island (ಉಡುಪಿ)**

---

## 2. Core Concepts (Concept-wise structured)

### 2.1 ಪರ್ವತಗಳು (Mountains)

#### Definition
- ಸುತ್ತಲಿನ ಪ್ರದೇಶಕ್ಕಿಂತ **ಎತ್ತರ**, **ಕಡಿದಾದ ಇಳಿಜಾರು**, **ಶಿಖರ** ಹೊಂದಿರುವ ನೈಸರ್ಗಿಕ ಭೂಸ್ವರೂಪ.
- ಸಾಮಾನ್ಯ ಮಾನದಂಡ:
  - **600 ಮೀ. ಕ್ಕಿಂತ ಕಡಿಮೆ** ಎತ್ತರದವು → **ಬೆಟ್ಟಗಳು (Hills)**
  - ಪರ್ವತದ ಅತ್ಯುನ್ನತ ತುದಿ → **ಶಿಖರ (Peak)**
  - ಸರಣಿಯಾಗಿ ಹರಡಿದ ಪರ್ವತಗಳ ಸಮೂಹ → **ಪರ್ವತ ಶ್ರೇಣಿ (Mountain Range)**

#### Key features
- ಎತ್ತರ + ಕಡಿದಾದ ಇಳಿಜಾರು + ಶಿಖರಗಳು
- ನದಿಗಳ ಉಗಮಸ್ಥಾನ, ಹವಾಮಾನ ನಿಯಂತ್ರಣ, ನೈಸರ್ಗಿಕ ಗಡಿ

#### Classification (Formation-based)
| ವಿಧ | ರಚನಾ ಪ್ರಕ್ರಿಯೆ | ಲಕ್ಷಣ | ಉದಾಹರಣೆಗಳು |
|---|---|---|---|
| **ಮಡಿಕೆ ಪರ್ವತಗಳು (Fold)** | ಆಂತರಿಕ ಒತ್ತಡ (Compression) → ಶಿಲಾಸ್ತರಗಳು ಮಡಿಕೆ | ಅತ್ಯಂತ ಎತ್ತರ; ಇತ್ತೀಚಿನ ಭೂವೈಜ್ಞಾನಿಕ ಯುಗದಲ್ಲಿ ರೂಪುಗೊಂಡವು | **ಹಿಮಾಲಯ**, **ಆಲ್ಪ್ಸ್** |
| **ಖಂಡ ಪರ್ವತಗಳು (Block)** | ಕರ್ಷಣ (Tension) → ಶಿಲಾಸ್ತರಭಂಗ (Faulting) → ಮಧ್ಯಭಾಗ ಮೇಲಕ್ಕೆ/ಸುತ್ತಲಿನ ಭಾಗ ಕುಸಿತ | Fold ಗಿಂತ ಕಡಿಮೆ ಎತ್ತರ | **Sierra Nevada**, **Vosges** |
| **ಜ್ವಾಲಾಮುಖಿ ಪರ್ವತಗಳು (Volcanic)** | ಜ್ವಾಲಾಮುಖಿ ಸ್ಫೋಟ → ಲಾವಾ/ಬೂದಿ ಸಂಚಯ → ಶಂಖಾಕೃತಿ | ವಿಸ್ತಾರ ತಳಹದಿ + ಶಂಖಾಕೃತಿಯ ಶಿಖರ | **Fujiyama**, **Kilimanjaro** |

#### Special category: ಪ್ರಾಚೀನ ಪರ್ವತಗಳು (Old Mountains)
- ದೀರ್ಘಕಾಲದ ಸವೆತದಿಂದ ಕಡಿಮೆ ಎತ್ತರ/ಮೃದುವಾದ ಶಿಖರಗಳು
- ಉದಾ: **ಅರಾವಳಿ**, **ಅಪಲೇಷಿಯನ್**

#### Important places/persons
- Places: Himalayas, Alps, Aravalli, Appalachians, Sierra Nevada, Vosges, Mt. Fuji, Kilimanjaro

#### Analytical points (Exam-use)
- **ಪರ್ವತಗಳು = ನೀರು + ಶಕ್ತಿ + ಗಡಿ + ಸಂಪನ್ಮೂಲ**
  - ನದಿಗಳ ಉಗಮ → **ನೀರಾವರಿ/ಕುಡಿಯುವ ನೀರು**
  - ಇಳಿಜಾರು/ಜಲಪಾತ → **ಜಲವಿದ್ಯುತ್**
  - ಹವಾಮಾನ ನಿಯಂತ್ರಣ → ಮಳೆ ಮಾದರಿ/ತಾಪಮಾನ
  - ಪ್ರವಾಸೋದ್ಯಮ + ತೋಟಗಾರಿಕೆ (plantation) + ಅರಣ್ಯ ಸಂಪತ್ತು

### 2.2 ಪ್ರಸ್ಥಭೂಮಿಗಳು (Plateaus)

#### Definition
- **ವಿಶಾಲ ಸಮತಟ್ಟಾದ ಮೇಲ್ಭಾಗ** + **ಕಡಿದಾದ ಇಳಿಜಾರು** ಹೊಂದಿರುವ, ಸುತ್ತಲಿನ ಪ್ರದೇಶಕ್ಕಿಂತ ಎತ್ತರದ ಭೂಸ್ವರೂಪ.
- ಪರ್ಯಾಯ ಹೆಸರು: **ಪೀಠಭೂಮಿ / Tableland**
- ಸಾಮಾನ್ಯವಾಗಿ **ಕಠಿಣ ಶಿಲೆಗಳು**, **ಖನಿಜ ಸಂಪನ್ಮೂಲ ಸಮೃದ್ಧ**

#### Key features
- ಸಮತಟ್ಟಾದ ಮೇಲ್ಭಾಗ (upland surface)
- ಖನಿಜ ಸಂಪತ್ತು, ಜಲಪಾತಗಳು, ಕೆಲವಡೆ ಲಾವಾ ಮಣ್ಣು

#### Classification (Location-based)
| ವಿಧ | ಸ್ಥಾನ/ಸುತ್ತುವರಿಕೆ | ಉದಾಹರಣೆ |
|---|---|---|
| **ಅಂತರಪರ್ವತ ಪ್ರಸ್ಥಭೂಮಿ (Intermontane)** | ಪರ್ವತ ಶ್ರೇಣಿಗಳಿಂದ ಸಂಪೂರ್ಣ ಆವೃತ | **ಟಿಬೆಟ್** (“Roof of the World”), **ಬೊಲಿವಿಯಾ** |
| **ಪರ್ವತಪಾದ ಪ್ರಸ್ಥಭೂಮಿ (Piedmont)** | ಒಂದು ಬದಿಯಲ್ಲಿ ಪರ್ವತ, ಇನ್ನೊಂದು ಬದಿಯಲ್ಲಿ ಮೈದಾನ/ಸಮುದ್ರ | **ಪಟಗೋನಿಯಾ** |
| **ಖಂಡಾಂತರ ಪ್ರಸ್ಥಭೂಮಿ (Continental)** | ಭೂಖಂಡದ ಭಾಗ ಮೇಲಕ್ಕೆತ್ತಲ್ಪಡುವುದು/ಲಾವಾ ಹರಡುವುದು | **ದಖನ್ ಪ್ರಸ್ಥಭೂಮಿ**, **ಆಫ್ರಿಕಾ ಪ್ರಸ್ಥಭೂಮಿ** |

#### Karnataka focus
- **ದಖನ್ ಪ್ರಸ್ಥಭೂಮಿ** ಭಾರತದ ದಕ್ಷಿಣ ಭಾಗದ ಪ್ರಮುಖ ಭೌತಿಕ ಘಟಕ; ಕರ್ನಾಟಕದ ಬಹುಭಾಗ ಇದರ ಭಾಗವಾಗಿ ಬರುತ್ತದೆ (ಪರೀಕ್ಷೆಯಲ್ಲಿ “Deccan Plateau—features/importance” ಸಾಮಾನ್ಯ).

#### Analytical points
- **ಖನಿಜ ಆಧಾರಿತ ಕೈಗಾರಿಕೆಗಳಿಗೆ ಮೂಲಭೂಮಿ**
  - ಚಿನ್ನ/ವಜ್ರ/ಕಬ್ಬಿಣ ಇತ್ಯಾದಿ (ಆಫ್ರಿಕಾ ಉದಾಹರಣೆ)
- **ಜಲವಿದ್ಯುತ್ ಸಾಧ್ಯತೆ**
  - ಕಡಿದಾದ ಇಳಿಜಾರು → ಜಲಪಾತಗಳು → hydel projects
- **ಕೃಷಿ/ಪಶುಸಂಗೋಪನೆ**
  - ಲಾವಾ ಮಣ್ಣು (ಕೆಲವು ಪ್ರದೇಶಗಳಲ್ಲಿ) → ಕೃಷಿಗೆ ಅನುಕೂಲ
  - ಹುಲ್ಲುಗಾವಲು → ಪಶುಸಂಗೋಪನೆ

### 2.3 ಮೈದಾನಗಳು (Plains)

#### Definition
- ವಿಸ್ತಾರವಾದ **ಸಮತಟ್ಟಾದ ಅಥವಾ ಅಲ್ಪ ಏರಿಳಿತಗಳಿರುವ** ತಗ್ಗು ಭೂಪ್ರದೇಶ.
- ಕೃಷಿ, ವಸತಿ, ಸಾರಿಗೆ, ನಗರೀಕರಣಕ್ಕೆ ಅತ್ಯಂತ ಅನುಕೂಲ → ಆರ್ಥಿಕ ಮಹತ್ವ ಹೆಚ್ಚು.

#### Key features
- ಸಮತಟ್ಟು ಭೂಮಿ → ಸಾರಿಗೆ/ಸಂಪರ್ಕ ಸುಲಭ
- ಸಂಚಯನ ಮೈದಾನಗಳಲ್ಲಿ ಫಲವತ್ತಾದ ಮೆಕ್ಕಲು ಮಣ್ಣು

#### Classification (Origin-based)
| ವಿಧ | ರಚನೆ | ಉದಾಹರಣೆ |
|---|---|---|
| **ರಚನಾ ಮೈದಾನ (Structural)** | ಸಮುದ್ರ ತಳ ಮೇಲಕ್ಕೆತ್ತಲ್ಪಡುವುದು / ಭೂಭಾಗ ಕುಸಿತ | USA ಆಗ್ನೇಯ ಭಾಗದ ಮೈದಾನ |
| **ಸವೆತ ಮೈದಾನ (Erosional)** | ನದಿ/ಹಿಮನದಿ/ಗಾಳಿ ಸವೆತದಿಂದ ಪರ್ವತ/ಪ್ರಸ್ಥಭೂಮಿ ನಯವಾಗುವುದು | ಪಶ್ಚಿಮ ಸೈಬೀರಿಯಾ |
| **ಸಂಚಯನ ಮೈದಾನ (Depositional)** | ನದಿ/ಹಿಮನದಿ/ಗಾಳಿ ತಂದ ಮೆಕ್ಕಲು/ಮರಳು ಸಂಚಯ | **ಸಿಂಧೂ–ಗಂಗಾ ಬಯಲು**, **ಸುಂದರಬನ್ಸ್ (ಮುಖಜಭೂಮಿ)** |

#### Analytical points
- “**ನಾಗರಿಕತೆಯ ತೊಟ್ಟಿಲು**” ಯಾಕೆ?
  - ಫಲವತ್ತಾದ ಮಣ್ಣು → ಕೃಷಿ ಅಧಿಕ ಉತ್ಪಾದನೆ
  - ಸಮತಟ್ಟು ನೆಲ → ರಸ್ತೆ/ರೈಲು/ನಗರ ವಿಸ್ತರಣೆ
  - ಜನಸಾಂದ್ರತೆ + ಮಾರುಕಟ್ಟೆ + ಕೈಗಾರಿಕೆಗಳ ಕೇಂದ್ರೀಕರಣ

---

### 2.4 ಮರುಭೂಮಿ ಮತ್ತು ದ್ವೀಪಗಳು (Deserts & Islands)

#### A) ಮರುಭೂಮಿ (Desert)

##### Definition
- ಅತ್ಯಂತ ಕಡಿಮೆ ಮಳೆ, ಶುಷ್ಕ, ಬಂಜರು ವಿಸ್ತಾರ ಪ್ರದೇಶ; **ಶಿಲಾಮಯ ಅಥವಾ ಮರಳುಮಯ**.
- ಮರುಭೂಮಿಯಲ್ಲಿ ಅಂತರ್ಜಲ ಚಿಲುಮೆಯಾಗಿ ಹೊರಬರುವ ಸ್ಥಳ → **ಓಯಸಿಸ್ (Oasis)**

##### Types
- **ಉಷ್ಣವಲಯದ ಮರುಭೂಮಿ**: ಹಗಲು ಅತಿ ಉಷ್ಣ, ರಾತ್ರಿ ತಂಪು
- **ಸಮಶೀತೋಷ್ಣವಲಯದ ಮರುಭೂಮಿ**

##### Examples
- **ಸಹರಾ** (ವಿಶ್ವದ ಅತಿದೊಡ್ಡ ಉಷ್ಣ ಮರುಭೂಮಿ)
- **ಥಾರ್** (ಭಾರತ)
- Oasis example: **ಈಜಿಪ್ಟ್—ನೈಲ್ ಕಣಿವೆ ಪ್ರದೇಶಗಳು** (ಪಾಠದಲ್ಲಿನ ಉದಾಹರಣೆ)

##### Analytical points
- ಕಠಿಣ ವಾಸಸ್ಥಾನ; ಆದರೆ **ಖನಿಜ/ಇಂಧನ ಸಂಪನ್ಮೂಲ** (ಉದಾ: ಸೌದಿ ಅರೇಬಿಯಾ—ಪೆಟ್ರೋಲಿಯಂ)
- Oasis → ವಸತಿ/ಕೃಷಿಗೆ ಕೇಂದ್ರ

---

#### B) ದ್ವೀಪ (Island)

##### Definition
- ಸಾಗರ/ಸಮುದ್ರ/ಸರೋವರ/ನದಿ ನೀರಿನಿಂದ ಸಂಪೂರ್ಣ ಆವೃತ ಭೂಭಾಗ.
- ಅತಿ ಚಿಕ್ಕ ದ್ವೀಪಗಳು → **ಕಿರುದ್ವೀಪ (Islets)**
- ದ್ವೀಪಗಳ ಸಮೂಹ → **ದ್ವೀಪಸ್ತೋಮ (Archipelago)**

##### Types
| ವಿಧ | ರಚನೆ/ಸ್ಥಿತಿ | ಉದಾಹರಣೆ |
|---|---|---|
| **ಖಂಡಾಂತರ ದ್ವೀಪ (Continental)** | ಪ್ರಧಾನ ಭೂಖಂಡದಿಂದ ಬೇರ್ಪಟ್ಟ | **Greenland**, **Iceland** |
| **ಸಾಗರಿಕ ದ್ವೀಪ (Oceanic)** | ಸಾಗರ ಮಧ್ಯದಲ್ಲಿ ರೂಪುಗೊಂಡ | (i) ಜ್ವಾಲಾಮುಖಿ (ii) ಹವಳದ |

**ಸಾಗರಿಕ ದ್ವೀಪಗಳ ಉಪಪ್ರಕಾರ**
- **ಜ್ವಾಲಾಮುಖಿ ದ್ವೀಪಗಳು**: ಸಾಗರ ತಳದ ಜ್ವಾಲಾಮುಖಿ ಚಟುವಟಿಕೆ
  - ಉದಾ: **Sumatra**, **St. Mary’s Island (Karnataka—ಉಡುಪಿ)**  
- **ಹವಳದ ದ್ವೀಪಗಳು**: coral polyps ಅಸ್ಥಿಪಂಜರ ಸಂಚಯ
  - ಉದಾ: **ಲಕ್ಷದ್ವೀಪ**

##### Karnataka focus
- **St. Mary’s Island (ಉಡುಪಿ)**: ಜ್ವಾಲಾಮುಖಿ ಮೂಲದ ದ್ವೀಪ ಎಂದು ಪಾಠದಲ್ಲಿ ಉಲ್ಲೇಖ—ನಕ್ಷೆ/ಸ್ಥಳ ಗುರುತಿಸುವ ಪ್ರಶ್ನೆಗೆ ಮುಖ್ಯ.

##### Analytical points
- ದ್ವೀಪಗಳು: ಪ್ರವಾಸೋದ್ಯಮ + ಮೀನುಗಾರಿಕೆ + ವಿಶಿಷ್ಟ ಪರಿಸರ ವ್ಯವಸ್ಥೆ
- ಹವಳದ ದ್ವೀಪಗಳು: ಸಮುದ್ರ ಜೀವವೈವಿಧ್ಯಕ್ಕೆ ಪ್ರಮುಖ; ಪರಿಸರ ಸಂರಕ್ಷಣೆ ಪ್ರಶ್ನೆಗಳಿಗೆ ಲಿಂಕ್

---

### 2.5 ಸ್ವಾಭಾವಿಕ ಪ್ರದೇಶಗಳು (Natural Regions)

#### Definition
- ಒಂದೇ ರೀತಿಯ **ವಾಯುಗುಣ (climate), ಮೇಲ್ಮೈ ಲಕ್ಷಣ, ಮಣ್ಣು, ಸಸ್ಯವರ್ಗ, ಪ್ರಾಣಿ ಸಂಕುಲ** ಹೊಂದಿರುವ ಭೂಮಿಯ ವಿಶಾಲ ಭಾಗ.
- ಗಡಿಗಳು **ನಿಖರವಾಗಿರದು**; ಕ್ರಮೇಣ ಬದಲಾವಣೆ (transition zones).

#### Key features
- Climate = ಪ್ರಮುಖ ನಿರ್ಣಾಯಕ ಅಂಶ (global classification)
- Comparative study framework: ಪರಿಸರ ↔ ಮಾನವ ಜೀವನಶೈಲಿ/ಕೃಷಿ/ಆರ್ಥಿಕ ಚಟುವಟಿಕೆ

#### Major natural regions (as per text)
1. ಸಮಭಾಜಕ ವೃತ್ತದ ಪ್ರದೇಶ (Amazon model)  
2. ಉಷ್ಣವಲಯದ ಹುಲ್ಲುಗಾವಲು (Sudan model)  
3. ಉಷ್ಣವಲಯದ ಮಾನ್ಸೂನ್ ಪ್ರದೇಶ  
4. ಉಷ್ಣವಲಯದ ಸಾಗರಿಕ ಪ್ರದೇಶ  
5. ಉಷ್ಣವಲಯದ ಮರುಭೂಮಿ (Sahara model)

## 5. Analytical Understanding (Very Important)

### A) Cause → Effect links (landforms)
- **Compression (ಆಂತರಿಕ ಒತ್ತಡ)** → ಶಿಲಾಸ್ತರ ಮಡಿಕೆ → **Fold mountains** → ಯುವ ಪರ್ವತಗಳು, ಹೆಚ್ಚಿನ ಎತ್ತರ
- **Tension + Faulting (ಕರ್ಷಣ + ಭಂಗ)** → ಭೂಭಾಗ ಮೇಲಕ್ಕೆ/ಕುಸಿತ → **Block mountains**
- **Volcanism** → ಲಾವಾ/ಬೂದಿ ಸಂಚಯ → **Volcanic mountains/islands**
- **River deposition** → ಮೆಕ್ಕಲು ಸಂಚಯ → **Depositional plains + deltas** → ಕೃಷಿ/ಜನಸಾಂದ್ರತೆ ಹೆಚ್ಚಳ
- **Erosion (river/glacier/wind)** → ಮೇಲ್ಮೈ ಸಮತಟ್ಟು → **Erosional plains** → ಕಡಿಮೆ ಎತ್ತರದ ಸಮತಟ್ಟು ಪ್ರದೇಶ

### B) Technology → Economy → Politics (exam-ready framing)
- Mountains/Plateaus → **hydel power potential** → ಕೈಗಾರಿಕೆ/ನಗರೀಕರಣಕ್ಕೆ ವಿದ್ಯುತ್ → ನೀರು ಹಂಚಿಕೆ/ಪರಿಸರ ನೀತಿ ಪ್ರಶ್ನೆಗಳು
- Plateaus → **minerals** → ಗಣಿಗಾರಿಕೆ/ಉದ್ಯಮ → ಕಾರ್ಮಿಕ ವಸತಿ/ಪ್ರಾದೇಶಿಕ ಅಭಿವೃದ್ಧಿ ಅಸಮಾನತೆ
- Plains → **transport corridors + agriculture surplus** → ನಗರ/ಮಾರುಕಟ್ಟೆ ಕೇಂದ್ರಗಳು → ಆಡಳಿತ/ರಾಜಕೀಯ ಕೇಂದ್ರಿಕರಣ (civilizational core)

### C) Social transformation points
- Plains: ಕೃಷಿ ಅಧಿಕ ಉತ್ಪಾದನೆ → ಜನಸಂಖ್ಯೆ ಸಾಂದ್ರತೆ → ನಗರೀಕರಣ/ವ್ಯಾಪಾರ
- Deserts: Oasis-ಕೇಂದ್ರಿತ ವಸತಿ → ಚದುರಿದ ಜನಸಂಖ್ಯೆ; ಸಂಪನ್ಮೂಲ (oil/minerals) ಬಂದರೆ ಆರ್ಥಿಕ ರೂಪಾಂತರ
- Islands: ಮೀನುಗಾರಿಕೆ/ಪ್ರವಾಸೋದ್ಯಮ ಆಧಾರಿತ ಆರ್ಥಿಕತೆ; ಪರಿಸರ ಸಂರಕ್ಷಣೆಯ ಒತ್ತಡ

### D) Evolution patterns (integrated system)
- ಪರ್ವತ → ನದಿ ಉಗಮ → ಮೈದಾನದಲ್ಲಿ ಸಂಚಯ → ಮುಖಜಭೂಮಿ/ಸಮುದ್ರ  
  (ಚಿತ್ರ/ರೇಖಾಚಿತ್ರ ಆಧಾರಿತ ಪ್ರಶ್ನೆಗಳ ಮೂಲ ತರ್ಕ)

- “ಸ್ವಾಭಾವಿಕ ಪ್ರದೇಶಗಳ ಅರ್ಥ, ಲಕ್ಷಣಗಳು ಮತ್ತು ಹವಾಮಾನದ ಪಾತ್ರ”

### C) MCQ traps (common)
- **Block vs Fold**: Block = faulting/tension; Fold = compression/folding
- **Depositional plain vs Erosional plain**: Depositional = fertile alluvium; Erosional = worn-down surface
- **Continental island vs Oceanic island**: Continental = separated from mainland; Oceanic = formed in ocean (volcanic/coral)
- **Oasis** is **in desert**, not in plains/plateaus
- **St. Mary’s Island**: island example (text says volcanic island) — Karnataka linkage

### D) Frequently confused points
- Plateau classification is **location-based** (intermontane/piedmont/continental), not by height.
- Natural regions have **no sharp boundaries**; they transition gradually.
- “Sundarbans” is **delta (depositional feature)**, not a mountain/plateau.
