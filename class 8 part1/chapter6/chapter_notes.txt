## 1. Chapter Overview

### Core Theme
- ದಕ್ಷಿಣ ಭಾರತದ ಪ್ರಮುಖ ಪ್ರಾಚೀನ ರಾಜವಂಶಗಳ **ರಾಜಕೀಯ ವಿಕಾಸ**, **ಆಡಳಿತ ವ್ಯವಸ್ಥೆ**, **ಸಾಮಾಜಿಕ-ಧಾರ್ಮಿಕ ಚಲನೆಗಳು**, **ಸಾಹಿತ್ಯ**, **ವಾಸ್ತುಶಿಲ್ಪ/ಶಿಲ್ಪಕಲೆ**ಗಳ ಸಮಗ್ರ ವಿಶ್ಲೇಷಣೆ.
- ಒಳಗೊಂಡ ರಾಜವಂಶಗಳು: **ಸಂಗಂ ಯುಗದ ಚೋಳ–ಚೇರ–ಪಾಂಡ್ಯ**, **ಶಾತವಾಹನ**, **ಕದಂಬ**, **ಪಶ್ಚಿಮ ಗಂಗ**, **ಬಾದಾಮಿ ಚಾಳುಕ್ಯ**, **ಪಲ್ಲವ**, **ರಾಷ್ಟ್ರಕೂಟ**, **ಕಲ್ಯಾಣ ಚಾಳುಕ್ಯ**, **ಹೊಯ್ಸಳ**, **ಚೋಳ (ಮಧ್ಯಯುಗೀನ)**.

### Why important for GPST / HSTR
- **Match the following / Capital–Dynasty / King–Title / Work–Author / Site–Dynasty** ಮಾದರಿಯ ಪ್ರಶ್ನೆಗಳಿಗೆ ಹೈ-ಯೀಲ್ಡ್.
- ಕರ್ನಾಟಕ ಇತಿಹಾಸದ “ಕೋರ್ ಲೈನ್”: **ಕದಂಬ → ಗಂಗ → ಚಾಳುಕ್ಯ → ರಾಷ್ಟ್ರಕೂಟ → ಕಲ್ಯಾಣ ಚಾಳುಕ್ಯ → ಹೊಯ್ಸಳ** (ವಿಜಯನಗರದ ಹಿನ್ನೆಲೆ).
- **ವಾಸ್ತುಶಿಲ್ಪ ಶೈಲಿ (ದ್ರಾವಿಡ/ವೇಸರ/ಹೊಯ್ಸಳ)**, **ಶಾಸನ-ಭಾಷೆ**, **ವಚನ ಚಳುವಳಿ**, **ಗ್ರಾಮಸ್ವರಾಜ್ಯ (ಚೋಳ)**—HSTR ಮಟ್ಟದ ವಿಶ್ಲೇಷಣೆಗೆ ಅತ್ಯಗತ್ಯ.

## 2. Core Concepts (Concept-wise structured)

### 2.1 ಸಂಗಂ ಯುಗ ಮತ್ತು ತಮಿಳು ಸಾಹಿತ್ಯ

#### Definition
- ಪಾಂಡ್ಯರ ರಾಜಧಾನಿ **ಮಧುರೈ**ಯಲ್ಲಿ ಕವಿಗಳು/ವಿದ್ವಾಂಸರು ಸೇರುವ **“ಸಂಗಂ”** ಸಭೆಗಳ ಆಶ್ರಯದಲ್ಲಿ ಬೆಳೆದ ಪ್ರಾಚೀನ ತಮಿಳು ಸಾಹಿತ್ಯಯುಗ.

#### Key Features
- ಸಾಹಿತ್ಯ = ಕಾವ್ಯ ಮಾತ್ರವಲ್ಲ; **ಸಮಾಜ–ಆರ್ಥಿಕತೆ–ರಾಜಕೀಯ–ಜನಜೀವನ**ದ ಐತಿಹಾಸಿಕ ದಾಖಲೆ.
- ಪ್ರಮುಖ ರಾಜವಂಶಗಳು: **ಚೋಳ, ಚೇರ, ಪಾಂಡ್ಯ**.

#### Important Works / Authors
- **ಶಿಲಪ್ಪದಿಗಾರಂ** — *ಇಳಂಗೋ ಅಡಿಗಳ್*  
  - ಪಾತಿವ್ರತ್ಯ, ನ್ಯಾಯ, ಸಾಮಾಜಿಕ ನೀತಿ.
- **ಮಣಿಮೇಖಲೈ** — ಬೌದ್ಧ ತತ್ವಪ್ರಧಾನ; *ಶಿಲಪ್ಪದಿಗಾರಂ*ನ ಮುಂದುವರಿಕೆ ಎಂದು ಪರಿಗಣನೆ.
- **ತಿರುಕ್ಕುರಳ್** — *ತಿರುವಳ್ಳುವರ್*  
  - **1330 ದ್ವಿಪದಿಗಳು**; ಧರ್ಮ–ಅರ್ಥ–ಕಾಮ; ವಿಶ್ವಮಾನ್ಯ ನೀತಿ ಸಂಹಿತೆ.

#### Analytical Points (Exam-use)
- ಸಂಗಂ ಸಾಹಿತ್ಯದಿಂದ:
  - ರಾಜ್ಯಗಳ ಪರಸ್ಪರ ಸಂಬಂಧ, ವ್ಯಾಪಾರ, ಧರ್ಮ, ಸಾಮಾಜಿಕ ಮೌಲ್ಯಗಳು ತಿಳಿಯುತ್ತವೆ.
  - ನಂತರದ ಚೋಳ ಸಾಮ್ರಾಜ್ಯಕ್ಕೆ **ಸಾಂಸ್ಕೃತಿಕ ಪರಂಪರೆಯ continuity**.

### 2.2 ಶಾತವಾಹನರು ಮತ್ತು ದಖ್ಖನ್ ಏಕೀಕರಣ

#### Definition
- ಮೌರ್ಯರ ನಂತರ **ದಖ್ಖನ್ ಪ್ರಸ್ಥಭೂಮಿಯಲ್ಲಿ** ವಿಶಾಲ ಸಾಮ್ರಾಜ್ಯ ನಿರ್ಮಿಸಿ ಉತ್ತರ–ದಕ್ಷಿಣ ಭಾರತ ನಡುವೆ **ಸಾಂಸ್ಕೃತಿಕ/ವಾಣಿಜ್ಯ ಸೇತುವೆ**ಯಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸಿದ ರಾಜವಂಶ.

#### Key Features
- **ರಾಜಧಾನಿ:** ಪ್ರತಿಷ್ಠಾನಪುರ (**ಪೈಠಾಣ**, ಮಹಾರಾಷ್ಟ್ರ)
- **ಆಡಳಿತ ಭಾಷೆ:** ಪ್ರಾಕೃತ
- **ಧಾರ್ಮಿಕ ನೀತಿ:** ವೈದಿಕ + ಬೌದ್ಧ ಸಮನ್ವಯ; ಬೌದ್ಧ ವಾಸ್ತುಶಿಲ್ಪಕ್ಕೆ ದೊಡ್ಡ ಕೊಡುಗೆ.

#### Important King / Titles
- **ಗೌತಮೀಪುತ್ರ ಶಾತಕರ್ಣಿ**
  - ಶಕ–ಯವನ–ಪಹ್ಲವರನ್ನು ಸೋಲಿಸಿದ.
  - ಬಿರುದು: **“ತ್ರಿಸಮುದ್ರತೋಯ ಪೀತವಾಹನ”**.

#### Literature
- **ಹಾಲ** — *ಗಾಥಾಸಪ್ತಶತಿ* (ಪ್ರಾಕೃತ)
- **ಗುಣಾಡ್ಯ** — *ವಡ್ಡಕಥಾ*

#### Architecture (Buddhist)
- **ಚೈತ್ಯಗಳು:** ಕಾರ್ಲೆ, ಕನ್ಹೇರಿ  
- **ವಿಹಾರಗಳು:** ಭಿಕ್ಷು ನಿವಾಸ
- **ಸ್ತೂಪಗಳು:** ಅಮರಾವತಿ (ಅಮೃತಶಿಲೆಯ ಸ್ತೂಪ)
- **ಕರ್ನಾಟಕ ಸಂಪರ್ಕ (ಪರೀಕ್ಷಾ ದೃಷ್ಟಿ):** **ಸನ್ನತಿ (ಕಲಬುರ್ಗಿ)**, **ಬನವಾಸಿ**

#### Economy
- **ರೋಮ್ ಜೊತೆ ವಾಣಿಜ್ಯ**
- ವ್ಯಾಪಾರಿ ಸಂಘಗಳು: **ಶ್ರೇಣಿ / ನಿಗಮ**

#### Analytical Points
- ವ್ಯಾಪಾರ ವೃದ್ಧಿ → **ನಗರೀಕರಣ** → ದಖ್ಖನ್ ಆರ್ಥಿಕ ಬುನಾದಿ.
- ಶಾತವಾಹನ ಪತನ → ಪ್ರಾದೇಶಿಕ ಶಕ್ತಿಗಳ ಉದಯ: **ಕದಂಬ (ಕರ್ನಾಟಕ), ಇಕ್ಷ್ವಾಕು (ಆಂಧ್ರ)**.

### 2.3 ಕನ್ನಡದ ಪ್ರಥಮ ರಾಜವಂಶಗಳು: ಕದಂಬರು ಮತ್ತು ಪಶ್ಚಿಮ ಗಂಗರು

#### Definition
- ಕರ್ನಾಟಕದಲ್ಲಿ **ಕನ್ನಡ/ಸಂಸ್ಕೃತಕ್ಕೆ ರಾಜಾಶ್ರಯ** ನೀಡಿ ಪ್ರಾದೇಶಿಕ ಅಸ್ಮಿತೆ ರೂಪಿಸಿದ ಮೊದಲ ಪ್ರಮುಖ ರಾಜವಂಶಗಳು.

---

#### A) ಕದಂಬರು

**Key Facts**
- **ಸ್ಥಾಪಕ:** **ಮಯೂರವರ್ಮ** (ಪಲ್ಲವರಿಂದ ಅವಮಾನ → ಕ್ಷತ್ರಿಯಧರ್ಮ ಸ್ವೀಕಾರ)
- **ರಾಜಧಾನಿ:** **ಬನವಾಸಿ (ಉತ್ತರ ಕನ್ನಡ)**

**Language / Epigraphy**
- ಕನ್ನಡದ ಅತಿ ಪ್ರಾಚೀನ ಶಾಸನ: **ಹಲ್ಮಿಡಿ ಶಾಸನ** (ಕದಂಬ ಕಾಲಕ್ಕೆ ಸೇರಿದೆ)

**Architecture**
- **ಬನವಾಸಿಯ ಮಧುಕೇಶ್ವರ ದೇವಾಲಯ**

**Analytical Point**
- ಕನ್ನಡವನ್ನು ಆಡಳಿತ/ಶಾಸನ ಮಟ್ಟಕ್ಕೆ ತರುವಲ್ಲಿ **ಮೊದಲ ಹೆಜ್ಜೆ**.

---

#### B) ಪಶ್ಚಿಮ ಗಂಗರು

**Key Facts**
- **ರಾಜಧಾನಿಗಳು:** ಕುವಲಾಲಪುರ (ಕೋಲಾರ) → ನಂತರ **ತಲಕಾಡು (ಮೈಸೂರು)**
- **ಪ್ರಮುಖ ಅರಸ:** **ದುರ್ವಿನೀತ** (ವಿದ್ವಾಂಸ + ಯೋಧ; *ಕಿರಾತಾರ್ಜುನೀಯ*ಕ್ಕೆ ಭಾಷ್ಯ)

**Literature / Scholars**
- ದುರ್ವಿನೀತ
- ಶ್ರೀಪುರುಷ — *ಗಜಶಾಸ್ತ್ರ*
- ಚಾವುಂಡರಾಯ — *ಚಾವುಂಡರಾಯ ಪುರಾಣ*

**Iconic Contribution**
- ಗಂಗರ ಮಂತ್ರಿ **ಚಾವುಂಡರಾಯ** ನಿರ್ಮಿಸಿದ:
  - **ಶ್ರವಣಬೆಳಗೊಳದ ಗೊಮ್ಮಟೇಶ್ವರ (58 ಅಡಿ) ಏಕಶಿಲಾ ವಿಗ್ರಹ**

**Analytical Point**
- ಗಂಗರು: ಜೈನಧರ್ಮ ಪೋಷಣೆ + ಶಿಲ್ಪ/ಸಾಹಿತ್ಯದಿಂದ ಕನ್ನಡ ಸಂಸ್ಕೃತಿ ಶ್ರೀಮಂತ.

### 2.4 ಚಾಳುಕ್ಯ–ಪಲ್ಲವ ಸಂಘರ್ಷ ಮತ್ತು ವಾಸ್ತುಶಿಲ್ಪದ ಉಗಮ

#### Definition
- **6ನೇ–8ನೇ ಶತಮಾನ**ಗಳಲ್ಲಿ ಬಾದಾಮಿ ಚಾಳುಕ್ಯರು ಮತ್ತು ಕಂಚಿಯ ಪಲ್ಲವರು ದಕ್ಷಿಣ ಭಾರತದ ಪ್ರಾಬಲ್ಯಕ್ಕಾಗಿ ನಡೆಸಿದ ಸಂಘರ್ಷ; ಇದರ ಪರಿಣಾಮವಾಗಿ **ವಾಸ್ತುಶಿಲ್ಪದಲ್ಲಿ ಸೃಜನಾತ್ಮಕ ಪೈಪೋಟಿ**.

#### Chalukyas of Badami (Vatapi)
- **ರಾಜಧಾನಿ:** ವಾತಾಪಿ (**ಬಾದಾಮಿ**)
- **ಪ್ರಮುಖ ಅರಸ:** **ಇಮ್ಮಡಿ ಪುಲಿಕೇಶಿ**
  - **ಹರ್ಷವರ್ಧನನನ್ನು ನರ್ಮದಾ ತೀರದಲ್ಲಿ ಸೋಲಿಸಿದ** (ದಕ್ಷಿಣಾಪಥದ ಸ್ವಾತಂತ್ರ್ಯ ಸೂಚಕ)
- **ವಾಸ್ತುಕೇಂದ್ರಗಳು (Karnataka focus):**
  - **ಐಹೊಳೆ** — “ಭಾರತೀಯ ದೇವಾಲಯ ವಾಸ್ತುಶಿಲ್ಪದ ತೊಟ್ಟಿಲು”
  - **ಬಾದಾಮಿ** — ಗುಹಾ ದೇವಾಲಯಗಳು
  - **ಪಟ್ಟದಕಲ್ಲು** — **UNESCO World Heritage Site**

#### Pallavas of Kanchi
- **ರಾಜಧಾನಿ:** **ಕಂಚಿ**
- **ಪ್ರಮುಖ ಅರಸ:** **ಒಂದನೇ ನರಸಿಂಹವರ್ಮ (ಮಹಾಮಲ್ಲ)**
  - ಪುಲಿಕೇಶಿಯನ್ನು ಸೋಲಿಸಿ ಬಿರುದು: **“ವಾತಾಪಿಕೊಂಡ”**
- **ವಾಸ್ತುಶಿಲ್ಪ:** **ಮಾಮಲ್ಲಪುರಂ (ಮಹಾಬಲಿಪುರಂ)**
  - ಏಕಶಿಲಾ ರಥಗಳು
  - **ಗಂಗಾವತರಣ** ಉಬ್ಬುಶಿಲ್ಪ
  - **Shore Temple**

#### Analytical Points
- ರಾಜಕೀಯ ಸ್ಪರ್ಧೆ → **ಕಲೆ/ವಾಸ್ತುಶಿಲ್ಪದಲ್ಲಿ innovation**
- ಚಾಳುಕ್ಯರು: ಉತ್ತರ–ದಕ್ಷಿಣ ಶೈಲಿ ಸಮ್ಮಿಳನ → **ವೇಸರ ಪ್ರವೃತ್ತಿಗೆ ನೆಲೆ**
- ಪಲ್ಲವರು: **ಏಕಶಿಲಾ ಶಿಲ್ಪ + ದೇವಾಲಯ ನಿರ್ಮಾಣ**ದಲ್ಲಿ ಮೈಲಿಗಲ್ಲು

### 2.5 ರಾಷ್ಟ್ರಕೂಟ ಸಾಮ್ರಾಜ್ಯ ಮತ್ತು ಸಾಂಸ್ಕೃತಿಕ ಸಾಧನೆಗಳು

#### Definition
- 8ನೇ ಶತಮಾನದಲ್ಲಿ ಚಾಳುಕ್ಯರನ್ನು ಪದಚ್ಯುತಗೊಳಿಸಿ ದಖ್ಖನ್‌ನಲ್ಲಿ ಪ್ರಭುತ್ವ ಸ್ಥಾಪಿಸಿದ ಅಖಿಲಭಾರತ ಮಟ್ಟದ ಶಕ್ತಿ.

#### Key Features
- **ರಾಜಧಾನಿ:** **ಮಾನ್ಯಖೇಟ (ಮಳಖೇಡ), ಕಲಬುರ್ಗಿ ಜಿಲ್ಲೆ**
- ಉತ್ತರ ಭಾರತದ ರಾಜಕಾರಣದಲ್ಲಿ ಪಾತ್ರ: **ಕನೌಜ್‌ಗಾಗಿ ತ್ರಿಕೋನ ಹೋರಾಟ** (Tripartite struggle)

#### Important Kings
- **ಮೂರನೆಯ ಗೋವಿಂದ** — ಹಿಮಾಲಯದವರೆಗೆ ದಿಗ್ವಿಜಯ
- **ಅಮೋಘವರ್ಷ ನೃಪತುಂಗ**
  - ದೀರ್ಘ ಆಳ್ವಿಕೆ; ವಿದ್ವಾಂಸ; **ಕವಿರಾಜಮಾರ್ಗ**ಕ್ಕೆ ಪ್ರೇರಕ
  - ಮಾನ್ಯಖೇಟ ನಿರ್ಮಾಣ
  - ಅರಬ್ ಪ್ರವಾಸಿ **ಸುಲೈಮಾನ್**: ವಿಶ್ವದ ನಾಲ್ಕು ಮಹಾಸಾಮ್ರಾಜ್ಯಗಳಲ್ಲಿ ಒಂದೆಂದು ವರ್ಣನೆ
- **ಮೂರನೆಯ ಕೃಷ್ಣ** — ಚೋಳರನ್ನು ಸೋಲಿಸಿ **ರಾಮೇಶ್ವರದವರೆಗೆ** ವಿಸ್ತರಣೆ

#### Literature (Kannada Golden Phase)
- **ಆದಿಕವಿ ಪಂಪ** — *ಆದಿಪುರಾಣ*, *ವಿಕ್ರಮಾರ್ಜುನ ವಿಜಯ*
- **ಶ್ರೀವಿಜಯ** — *ಕವಿರಾಜಮಾರ್ಗ* (ಕನ್ನಡದ ಮೊದಲ ಲಭ್ಯ ಲಕ್ಷಣ ಗ್ರಂಥ)
- **ಪೊನ್ನ** — *ಶಾಂತಿಪುರಾಣ*

#### Architecture
- **ಎಲ್ಲೋರಾ ಕೈಲಾಸನಾಥ ದೇವಾಲಯ** — ಒಂದನೇ ಕೃಷ್ಣ; ವಿಶ್ವದ ಅತಿದೊಡ್ಡ ಏಕಶಿಲಾ ರಚನೆ
- **ಎಲಿಫೆಂಟಾ ಗುಹೆಗಳು** — ‘ಮಹೇಶಮೂರ್ತಿ’ (ತ್ರಿಮೂರ್ತಿ) ಪ್ರಸಿದ್ಧ

#### Analytical Points
- ರಾಷ್ಟ್ರಕೂಟರು: **ರಾಜಕೀಯ ವಿಸ್ತರಣೆ + ಕನ್ನಡ ಸಾಹಿತ್ಯ/ಶಿಲ್ಪ** ಎರಡನ್ನೂ ಉತ್ತುಂಗಕ್ಕೆ ಕೊಂಡೊಯ್ದರು.
- ಪತನ ನಂತರ: **ಕಲ್ಯಾಣದ ಚಾಳುಕ್ಯರು** ಪುನರುತ್ಥಾನ.

---

### 2.6 ಕಲ್ಯಾಣದ ಚಾಳುಕ್ಯರು (ಪಶ್ಚಿಮ/ನಂತರದ ಚಾಳುಕ್ಯರು) ಮತ್ತು ವಚನ ಚಳುವಳಿ

#### Definition
- ರಾಷ್ಟ್ರಕೂಟರ ನಂತರ ಅಧಿಕಾರಕ್ಕೆ ಬಂದ ರಾಜವಂಶ; ಕರ್ನಾಟಕದಲ್ಲಿ **ವಚನ ಸಾಹಿತ್ಯ/ಸಾಮಾಜಿಕ ಸುಧಾರಣೆ**ಯುಗಕ್ಕೆ ವೇದಿಕೆ.

#### Key Facts
- **ರಾಜಧಾನಿ:** **ಬಸವಕಲ್ಯಾಣ (ಬೀದರ್ ಜಿಲ್ಲೆ)**

#### Important Kings / Scholars
- **ಆರನೆಯ ವಿಕ್ರಮಾದಿತ್ಯ**
  - **ಚಾಳುಕ್ಯ ವಿಕ್ರಮ ಶಕೆ** ಆರಂಭ
  - ಆಸ್ಥಾನ:
    - **ಬಿಲ್ಹಣ** — *ವಿಕ್ರಮಾಂಕದೇವಚರಿತ*
    - **ವಿಜ್ಞಾನೇಶ್ವರ** — *ಮಿತಾಕ್ಷರ* (ಹಿಂದೂ ಉತ್ತರಾಧಿಕಾರ ಕಾನೂನಿಗೆ ಮೂಲಗ್ರಂಥ)
- **ಮೂರನೆಯ ಸೋಮೇಶ್ವರ**
  - *ಮಾನಸೋಲ್ಲಾಸ / ಅಭಿಲಷಿತಾರ್ಥ ಚಿಂತಾಮಣಿ* (ಸಂಸ್ಕೃತ ವಿಶ್ವಕೋಶ)
  - ಬಿರುದು: **ಸರ್ವಜ್ಞ ಚಕ್ರವರ್ತಿ**

#### Literature
- **ರನ್ನ (ಕವಿಚಕ್ರವರ್ತಿ)** — *ಸಾಹಸಭೀಮ ವಿಜಯ (ಗದಾಯುದ್ಧ)*, *ಅಜಿತನಾಥ ಪುರಾಣ*
- **ರತ್ನತ್ರಯರು:** **ಪಂಪ–ಪೊನ್ನ–ರನ್ನ**

#### Vachana Movement (High analytical)
- ಸ್ವರೂಪ: ಸರಳ ಕನ್ನಡದಲ್ಲಿ ಅನುಭವಾಧಾರಿತ ಸಾಹಿತ್ಯ; ಸಾಮಾಜಿಕ–ಧಾರ್ಮಿಕ ಸುಧಾರಣೆ.
- ಪ್ರಮುಖ ವಚನಕಾರರು: **ಬಸವಣ್ಣ, ಅಲ್ಲಮಪ್ರಭು, ಅಕ್ಕಮಹಾದೇವಿ**
- ತತ್ವಗಳು: **ಕಾಯಕ**, **ದಾಸೋಹ**, ಜಾತಿ/ಲಿಂಗ ಭೇದ ವಿರೋಧ.

#### Architecture
- ಚಾಳುಕ್ಯ–ಹೊಯ್ಸಳ ಶೈಲಿಗಳ ಮಧ್ಯದ “ಸೇತುವೆ”
- **ಇಟಗಿಯ ಮಹಾದೇವ ದೇವಾಲಯ** — “**ದೇವಾಲಯಗಳ ಚಕ್ರವರ್ತಿ**” (ಶಾಸನೋಕ್ತ)

---

### 2.7 ಹೊಯ್ಸಳರು ಮತ್ತು ಶಿಲ್ಪಕಲೆಯ ಪರಾಕಾಷ್ಠೆ

#### Definition
- ಕಲ್ಯಾಣ ಚಾಳುಕ್ಯರ ನಂತರ ಕರ್ನಾಟಕದಲ್ಲಿ ಪ್ರಬಲ; **ಬಳಪದ ಕಲ್ಲಿನ (Soapstone) ಸೂಕ್ಷ್ಮ ಶಿಲ್ಪ**ದಿಂದ ವಿಶ್ವವಿಖ್ಯಾತ.

#### Key Facts
- **ರಾಜಧಾನಿ:** **ದ್ವಾರಸಮುದ್ರ (ಹಳೇಬೀಡು)**

#### Important Kings / Titles
- **ವಿಷ್ಣುವರ್ಧನ**
  - ಚೋಳರಿಂದ **ತಲಕಾಡು** ಗೆದ್ದು ಬಿರುದು: **“ತಲಕಾಡುಗೊಂಡ”**
- **ಮೂರನೆಯ ಬಲ್ಲಾಳ**
  - ದೆಹಲಿ ಸುಲ್ತಾನರ ದಾಳಿಗಳನ್ನು ಎದುರಿಸಿದ ಕೊನೆಯ ಪ್ರಮುಖ ಹೊಯ್ಸಳ ಅರಸ

#### Hoysala Architecture: Key Features (Must-remember)
- **ನಕ್ಷತ್ರಾಕಾರದ ಜಗತಿ (platform)**
- **Soapstone** ಬಳಕೆ → ಸೂಕ್ಷ್ಮ ಕೆತ್ತನೆ ಸಾಧ್ಯ
- ಹೊರಗೋಡೆಗಳ ಪಟ್ಟಿಕೆ ಕೆತ್ತನೆ: ಆನೆ–ಕುದುರೆ–ಲತಾಬಳ್ಳಿ–ಪುರಾಣ ಕಥೆಗಳು
- **ಮದನಿಕೆ (ಶಿಲಾಬಾಲಿಕೆ)** ವಿಗ್ರಹಗಳು
- ಛಾವಣಿಯ ಅಲಂಕಾರ: **ಭುವನೇಶ್ವರಿ**

#### Major Temples (Karnataka map focus)
- **ಬೇಲೂರು** — ಚೆನ್ನಕೇಶವ
- **ಹಳೇಬೀಡು** — ಹೊಯ್ಸಳೇಶ್ವರ
- **ಸೋಮನಾಥಪುರ** — ಕೇಶವ (UNESCO World Heritage Site)

#### Literature
- **ಜನ್ನ** — *ಯಶೋಧರ ಚರಿತೆ*
- **ಹರಿಹರ** — ರಗಳೆ ಸಾಹಿತ್ಯ
- **ರಾಘವಾಂಕ** — ಷಟ್ಪದಿ

#### Analytical Points
- ಹೊಯ್ಸಳ ದೇವಾಲಯ = “ಕಲ್ಲಿನಲ್ಲಿ ಅರಳಿದ ಕಾವ್ಯ”: ಧಾರ್ಮಿಕ ಕಥೆಗಳು + ಸಾಮಾಜಿಕ ಜೀವನದ ಪ್ರತಿಬಿಂಬ.
- ಹೊಯ್ಸಳ ಪತನ/ರಾಜಕೀಯ ಶೂನ್ಯತೆ → **ವಿಜಯನಗರ ಉದಯಕ್ಕೆ ಹಿನ್ನೆಲೆ**.

---

### 2.8 ಚೋಳ ಸಾಮ್ರಾಜ್ಯ: ಕಡಲಾಚೆಯ ಪ್ರಭುತ್ವ ಮತ್ತು ಸ್ಥಳೀಯ ಆಡಳಿತ

#### Definition
- 9ನೇ ಶತಮಾನದಲ್ಲಿ ಪುನರುತ್ಥಾನಗೊಂಡ ಚೋಳರು: ದಕ್ಷಿಣ ಭಾರತ + ಕಡಲಾಚೆ ಪ್ರದೇಶಗಳಲ್ಲಿ ಸಾಮ್ರಾಜ್ಯ; **ನೌಕಾಪಡೆ**, **ದ್ರಾವಿಡ ದೇವಾಲಯಗಳು**, **ಗ್ರಾಮಸ್ವರಾಜ್ಯ**ಕ್ಕೆ ಪ್ರಸಿದ್ಧ.

#### Key Facts
- **ರಾಜಧಾನಿ:** **ತಂಜಾವೂರು**

#### Important Kings / Achievements
- **ರಾಜರಾಜ ಚೋಳ I**
  - ಚೋಳ ಸಾಮ್ರಾಜ್ಯದ ನಿಜವಾದ ಸ್ಥಾಪಕ
  - **ಶ್ರೀಲಂಕಾ, ಮಾಲ್ಡೀವ್ಸ್** ಗೆ ಜಯ
  - **ತಂಜಾವೂರು ಬೃಹದೀಶ್ವರ ದೇವಾಲಯ** ನಿರ್ಮಾಣ (UNESCO)
- **ರಾಜೇಂದ್ರ ಚೋಳ I**
  - ಗಂಗಾ ತನಕ ದಿಗ್ವಿಜಯ → ಬಿರುದು: **“ಗಂಗೈಕೊಂಡ”**
  - **ಶ್ರೀವಿಜಯ (ಸುಮಾತ್ರಾ)** ಮೇಲೆ ನೌಕಾ ದಾಳಿ
  - ಹೊಸ ರಾಜಧಾನಿ: **ಗಂಗೈಕೊಂಡ ಚೋಳಪುರಂ**

#### Local Administration (High-yield governance)
- **ಊರ್** — ಸಾಮಾನ್ಯ ಗ್ರಾಮಸಭೆ  
- **ಸಭೆ** — ಬ್ರಾಹ್ಮಣ ಅಗ್ರಹಾರಗಳ ಸಭೆ  
- ಸದಸ್ಯ ಆಯ್ಕೆ: **ಕುಡವೋಲೈ (lottery) ಪದ್ಧತಿ**
- ಸಮಿತಿಗಳ ಮೂಲಕ ಕಾರ್ಯನಿರ್ವಹಣೆ + **ಕಡ್ಡಾಯ ಲೆಕ್ಕಪತ್ರ ಪರಿಶೋಧನೆ**
- ಗ್ರಾಮಗಳು: “**Little Republics**” ಎಂದು ವರ್ಣನೆ (ಆಧುನಿಕ ಪ್ರಜಾಪ್ರಭುತ್ವದ ಸಮೀಪದ ಲಕ್ಷಣಗಳು)

#### Art & Architecture
- **ಕಂಚಿನ ವಿಗ್ರಹಗಳು:** **ನಟರಾಜ** — ಚೋಳ ಕಂಚುಶಿಲ್ಪದ ಶಿಖರ
- **ದ್ರಾವಿಡ ಶೈಲಿ:** ಬೃಹದೀಶ್ವರ ದೇವಾಲಯ — ಪರಾಕಾಷ್ಠೆ

#### Literature
- **ಕಂಬನ್** — *ಕಂಬ ರಾಮಾಯಣಂ*

#### Analytical Points
- ನೌಕಾಪಡೆ/ವಾಣಿಜ್ಯ ನಿಯಂತ್ರಣ → ಹಿಂದೂ ಮಹಾಸಾಗರದಲ್ಲಿ ಪ್ರಭುತ್ವ → ರಾಜಕೀಯ ವಿಸ್ತರಣೆ.
- ಸ್ಥಳೀಯ ಸ್ವರಾಜ್ಯ → ಆಡಳಿತ ದಕ್ಷತೆ + ತೆರಿಗೆ/ಲೆಕ್ಕಪತ್ರ ಪಾರದರ್ಶಕತೆ.

## 4. Important Places (Karnataka focus + map utility)

### Karnataka: Must-mark Sites
- **ಬನವಾಸಿ (ಉತ್ತರ ಕನ್ನಡ)** — ಕದಂಬ ರಾಜಧಾನಿ; ಮಧುಕೇಶ್ವರ ದೇವಾಲಯ
- **ಹಲ್ಮಿಡಿ (ಹಾಸನ ಜಿಲ್ಲೆ)** — ಪ್ರಾಚೀನ ಕನ್ನಡ ಶಾಸನ (ಹಲ್ಮಿಡಿ ಶಾಸನ)
- **ತಲಕಾಡು (ಮೈಸೂರು)** — ಪಶ್ಚಿಮ ಗಂಗ ರಾಜಧಾನಿ; ಹೊಯ್ಸಳ–ಚೋಳ ಸಂಘರ್ಷ ಹಿನ್ನೆಲೆ
- **ಶ್ರವಣಬೆಳಗೊಳ (ಹಾಸನ)** — ಗೊಮ್ಮಟೇಶ್ವರ (ಚಾವುಂಡರಾಯ; ಗಂಗ)
- **ಐಹೊಳೆ (ಬಾಗಲಕೋಟೆ)** — “ವಾಸ್ತುಶಿಲ್ಪದ ತೊಟ್ಟಿಲು” (ಚಾಳುಕ್ಯ)
- **ಬಾದಾಮಿ (ಬಾಗಲಕೋಟೆ)** — ವಾತಾಪಿ; ಗುಹಾ ದೇವಾಲಯಗಳು
- **ಪಟ್ಟದಕಲ್ಲು (ಬಾಗಲಕೋಟೆ)** — UNESCO; ಚಾಳುಕ್ಯ ವಾಸ್ತು
- **ಇಟಗಿ (ಕೊಪ್ಪಳ)** — ಮಹಾದೇವ ದೇವಾಲಯ (“ದೇವಾಲಯಗಳ ಚಕ್ರವರ್ತಿ”)
- **ಮಳಖೇಡ/ಮಾನ್ಯಖೇಟ (ಕಲಬುರ್ಗಿ)** — ರಾಷ್ಟ್ರಕೂಟ ರಾಜಧಾನಿ
- **ಸನ್ನತಿ (ಕಲಬುರ್ಗಿ)** — ಶಾತವಾಹನ/ಬೌದ್ಧ ಪರಂಪರೆಯ ಸಂಪರ್ಕ
- **ಬಸವಕಲ್ಯಾಣ (ಬೀದರ್)** — ಕಲ್ಯಾಣ ಚಾಳುಕ್ಯ ರಾಜಧಾನಿ; ವಚನ ಚಳುವಳಿ ಹಿನ್ನೆಲೆ
- **ಬೇಲೂರು, ಹಳೇಬೀಡು, ಸೋಮನಾಥಪುರ (ಹಾಸನ/ಮೈಸೂರು ವಲಯ)** — ಹೊಯ್ಸಳ ದೇವಾಲಯಗಳು (ಸೋಮನಾಥಪುರ UNESCO)

### Outside Karnataka (Map-based)
- **ಪೈಠಾಣ** (ಮಹಾರಾಷ್ಟ್ರ) — ಶಾತವಾಹನ
- **ಎಲ್ಲೋರಾ** (ಮಹಾರಾಷ್ಟ್ರ) — ಕೈಲಾಸನಾಥ (ರಾಷ್ಟ್ರಕೂಟ)
- **ಎಲಿಫೆಂಟಾ** (ಮುಂಬೈ ಬಳಿ) — ಮಹೇಶಮೂರ್ತಿ
- **ಕಂಚಿ, ಮಾಮಲ್ಲಪುರಂ** (ತಮಿಳುನಾಡು) — ಪಲ್ಲವ
- **ತಂಜಾವೂರು, ಗಂಗೈಕೊಂಡ ಚೋಳಪುರಂ** (ತಮಿಳುನಾಡು) — ಚೋಳ

> **Map-pointing tip:** ರಾಜಧಾನಿಗಳು + ವಾಸ್ತುಕೇಂದ್ರಗಳನ್ನು ಜೋಡಿಸಿ ಓದಿ:  
> **ಬಾದಾಮಿ–ಐಹೊಳೆ–ಪಟ್ಟದಕಲ್ಲು** (ಚಾಳುಕ್ಯ ಕ್ಲಸ್ಟರ್), **ಬೇಲೂರು–ಹಳೇಬೀಡು–ಸೋಮನಾಥಪುರ** (ಹೊಯ್ಸಳ ಟ್ರೈಯಾಡ್), **ಮಾನ್ಯಖೇಟ–ಸನ್ನತಿ** (ಕಲಬುರ್ಗಿ ವಲಯ).

## 5. Analytical Understanding (Very Important)

### A. Cause → Effect Links
- **ಶಾತವಾಹನರ ರೋಮ್ ವ್ಯಾಪಾರ** → ಹಣ/ನಗರೀಕರಣ → ದಖ್ಖನ್ ಆರ್ಥಿಕ ಬಲ → ನಂತರದ ಪ್ರಾದೇಶಿಕ ರಾಜ್ಯಗಳ ಬೆಳವಣಿಗೆ.
- **ಚಾಳುಕ್ಯ–ಪಲ್ಲವ ರಾಜಕೀಯ ಸ್ಪರ್ಧೆ** → ವಾಸ್ತುಶಿಲ್ಪದಲ್ಲಿ ಪ್ರಯೋಗ/ಪೈಪೋಟಿ → ದ್ರಾವಿಡ/ವೇಸರ ಶೈಲಿಗಳ ವಿಕಾಸ.
- **ರಾಷ್ಟ್ರಕೂಟರ ಅಖಿಲಭಾರತ ರಾಜಕೀಯ (ತ್ರಿಕೋನ ಹೋರಾಟ)** → ದಖ್ಖನ್ ಶಕ್ತಿಯ ರಾಷ್ಟ್ರೀಯ ಮಹತ್ವ ಸ್ಥಾಪನೆ.
- **ವಚನ ಚಳುವಳಿ** → ಸಾಮಾಜಿಕ ಸಮಾನತೆ/ಕಾಯಕ-ದಾಸೋಹ ತತ್ವ → ಕರ್ನಾಟಕದ ಧಾರ್ಮಿಕ-ಸಾಮಾಜಿಕ ಪರಿವರ್ತನೆ.
- **ಹೊಯ್ಸಳರ ಕ್ಷೀಣತೆ + ದೆಹಲಿ ಸುಲ್ತಾನ ದಾಳಿಗಳು** → ರಾಜಕೀಯ ಶೂನ್ಯತೆ → **ವಿಜಯನಗರ ಉದಯ**ಕ್ಕೆ ನೆಲೆ.

### B. Technology → Economy → Politics (Interlink)
- **ಶಿಲ್ಪ/ವಾಸ್ತು ತಂತ್ರಜ್ಞಾನ** (ಗುಹಾ ಕತ್ತರಿಕೆ, ಏಕಶಿಲಾ, soapstone carving)  
  → ದೇವಾಲಯ ಕೇಂದ್ರಿತ ಆರ್ಥಿಕತೆ/ಕಲಾವೃತ್ತಿಗಳು  
  → ರಾಜಸತ್ತೆಯ ಪ್ರತಿಷ್ಠೆ/ಧಾರ್ಮಿಕ legitimation.
- **ಚೋಳ ನೌಕಾಪಡೆ**  
  → ಸಮುದ್ರ ವ್ಯಾಪಾರ ಮಾರ್ಗ ನಿಯಂತ್ರಣ  
  → ಕಡಲಾಚೆ ಸಾಮ್ರಾಜ್ಯ ವಿಸ್ತರಣೆ (ಶ್ರೀವಿಜಯ ದಾಳಿ).

### C. Social Transformation Patterns
- ಸಂಗಂ ಸಾಹಿತ್ಯ: ಸಮಾಜದ ಮೌಲ್ಯಗಳು/ನ್ಯಾಯ ಕಲ್ಪನೆಗಳ ದಾಖಲೆ.
- ವಚನ ಸಾಹಿತ್ಯ: **ಭಕ್ತಿ + ಸಾಮಾಜಿಕ ನ್ಯಾಯ**; ಭಾಷೆ democratization (ಸರಳ ಕನ್ನಡ).
- ಚೋಳ ಗ್ರಾಮಸಭೆಗಳು: **ಸ್ಥಳೀಯ ಸಂಸ್ಥೆಗಳ ಬಲ** → ಆಡಳಿತದಲ್ಲಿ ಭಾಗವಹಿಸುವಿಕೆ.

### D. Evolution Patterns (Continuities)
- **ಕದಂಬ–ಗಂಗ**: ಕನ್ನಡ ರಾಜಾಶ್ರಯದ ಆರಂಭ/ಬಲವರ್ಧನೆ  
- **ರಾಷ್ಟ್ರಕೂಟ–ಕಲ್ಯಾಣ ಚಾಳುಕ್ಯ–ಹೊಯ್ಸಳ**: ಸಾಹಿತ್ಯ/ವಾಸ್ತು ಪರಂಪರೆಯ ಕ್ರಮಿಕ ವಿಕಾಸ  
- **ಪಲ್ಲವ → ಚೋಳ**: ತಮಿಳುನಾಡಿನಲ್ಲಿ ದೇವಾಲಯ ಪರಂಪರೆಯ ಮುಂದುವರಿಕೆ

## 6. Previous Exam Focus Areas

### A) 1–2 Mark Facts (Direct)
- ಶಾತವಾಹನರ ರಾಜಧಾನಿ: **ಪೈಠಾಣ (ಪ್ರತಿಷ್ಠಾನಪುರ)**
- ಕದಂಬರ ರಾಜಧಾನಿ: **ಬನವಾಸಿ**
- ಪಶ್ಚಿಮ ಗಂಗರ ರಾಜಧಾನಿ: **ತಲಕಾಡು** (ನಂತರ)
- ಬಾದಾಮಿ ಚಾಳುಕ್ಯರ ರಾಜಧಾನಿ: **ವಾತಾಪಿ (ಬಾದಾಮಿ)**
- ಪಲ್ಲವರ ರಾಜಧಾನಿ: **ಕಂಚಿ**
- ರಾಷ್ಟ್ರಕೂಟರ ರಾಜಧಾನಿ: **ಮಾನ್ಯಖೇಟ (ಮಳಖೇಡ)**
- ಕಲ್ಯಾಣ ಚಾಳುಕ್ಯರ ರಾಜಧಾನಿ: **ಬಸವಕಲ್ಯಾಣ**
- ಹೊಯ್ಸಳರ ರಾಜಧಾನಿ: **ದ್ವಾರಸಮುದ್ರ (ಹಳೇಬೀಡು)**
- ಚೋಳರ ರಾಜಧಾನಿ: **ತಂಜಾವೂರು**
- “ವಾತಾಪಿಕೊಂಡ” ಬಿರುದು: **ನರಸಿಂಹವರ್ಮ (ಮಹಾಮಲ್ಲ)**
- “ಗಂಗೈಕೊಂಡ” ಬಿರುದು: **ರಾಜೇಂದ್ರ ಚೋಳ I**
- ಕನ್ನಡದ ಮೊದಲ ಲಭ್ಯ ಲಕ್ಷಣಗ್ರಂಥ: **ಕವಿರಾಜಮಾರ್ಗ**
- ಪ್ರಾಚೀನ ಕನ್ನಡ ಶಾಸನ: **ಹಲ್ಮಿಡಿ ಶಾಸನ**
- “ದೇವಾಲಯಗಳ ಚಕ್ರವರ್ತಿ”: **ಇಟಗಿಯ ಮಹಾದೇವ ದೇವಾಲಯ**
