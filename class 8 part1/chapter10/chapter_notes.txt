## 1. Chapter Overview

### Core Theme
- ಭೂಗೋಳಶಾಸ್ತ್ರದಲ್ಲಿ ಭೂಮಿಯ ಮೇಲ್ಮೈ ಲಕ್ಷಣಗಳನ್ನು ಪ್ರತಿನಿಧಿಸುವ **ಎರಡು ಪ್ರಮುಖ ಉಪಕರಣಗಳು**:
  - **ಗ್ಲೋಬ್ (Globe / ಮಾದರಿ ಗೋಳ)** – 3D, ಅವಿಕೃತ (without distortion) ಪ್ರತಿನಿಧಿ
  - **ನಕ್ಷೆ (Map)** – 2D, ಮಾನಕ (Scale) ಆಧಾರಿತ ಸಮತಟ್ಟಾದ ಪ್ರತಿನಿಧಿ
- **ನಕ್ಷೆ ಓದುವಿಕೆ (Map Reading)** ಮತ್ತು **ನಕ್ಷೆ ವ್ಯಾಖ್ಯಾನ (Map Interpretation)** ಗೆ ಅಗತ್ಯವಾದ ಮೂಲ ಜ್ಞಾನ:  
  **ಮಾನಕ, ದಿಕ್ಕು, ಅಕ್ಷಾಂಶ-ರೇಖಾಂಶ ಜಾಲ, ಸೂಚಿ/ಕೀ, ಚಿಹ್ನೆಗಳು-ಸಂಕೇತಗಳು**

### Why important for GPST / HSTR
- GPST/HSTR ನಲ್ಲಿ:
  - **ನಕ್ಷೆಯ ಮೂಲಾಂಶಗಳು, ಮಾನಕ, ಚಿಹ್ನೆಗಳು** → 1–2 ಅಂಕಗಳ ನೇರ ಪ್ರಶ್ನೆಗಳು
  - **Large vs Small scale**, **Globe vs Map** → MCQ ಟ್ರ್ಯಾಪ್‌ಗಳು
  - **Karnataka political map + distance calculation (graphical scale)** → ಅನ್ವಯಿಕ ಪ್ರಶ್ನೆಗಳು
- ಪಠ್ಯಕ್ರಮದ ಮುಂದಿನ ಘಟಕಗಳಲ್ಲಿ (ಭೌತಿಕ/ರಾಜಕೀಯ/ವಿಷಯಾಧಾರಿತ ನಕ್ಷೆಗಳು, GIS) ಅಡಿಪಾಯ.

## 2. Core Concepts (Concept-wise structured)

### Concept 1: Globe (ಗ್ಲೋಬ್ / ಮಾದರಿ ಗೋಳ)

#### Definition
- ಭೂಮಿಯ **Geoid** ಆಕಾರವನ್ನು ಪ್ರತಿನಿಧಿಸುವ **ಮೂರು ಆಯಾಮದ (3D) ಗೋಳಾಕಾರದ ಮಾದರಿ**.
- ಖಂಡ-ಸಾಗರಗಳ **ಗಾತ್ರ, ಆಕಾರ, ಪರಸ್ಪರ ಸ್ಥಾನ**ಗಳನ್ನು **ವಿಕೃತೀಕರಣವಿಲ್ಲದೆ** ತೋರಿಸುತ್ತದೆ.

#### Key Features
- **ಭೂಮಿಯ ಅಕ್ಷದ ಓರೆ:** 23.5° tilt ನ ನಿಖರ ಪ್ರದರ್ಶನ
- **ಅಕ್ಷಾಂಶ–ರೇಖಾಂಶ ಜಾಲ:** ಸ್ಥಳಗಳ absolute location ತಿಳಿಯಲು ಸಹಾಯಕ
- **ಚಲನೆಗಳ ಪ್ರದರ್ಶನ:**  
  - ಪರಿಭ್ರಮಣೆ (Rotation) → ಹಗಲು-ರಾತ್ರಿ  
  - ಪರಿಕ್ರಮಣೆ (Revolution) → ಋತುಮಾನಗಳು
- **ಸಮಗ್ರ ದೃಷ್ಟಿ:** ಭೂಮಿಯ ಮೇಲ್ಮೈಯನ್ನು ತಿರುಗಿಸಿ ವೀಕ್ಷಿಸಬಹುದು

#### Important Analytical Points
- **Strength:** distortion ಇಲ್ಲದ ಸಮಗ್ರ ಭೌಗೋಳಿಕ ಅರಿವು
- **Limitation:**  
  - **Small scale** → ವಿವರ ಕಡಿಮೆ  
  - ಸಾಗಿಸಲು ಅನಾನುಕೂಲ  
  - ಸ್ಥಳೀಯ/ಪ್ರಾದೇಶಿಕ ಅಧ್ಯಯನಕ್ಕೆ ಸೂಕ್ತವಲ್ಲ

#### Example (Exam-use)
- ಭಾರತ–USA ನಡುವೆ **ಅತಿ ಕಡಿಮೆ ವಾಯುಯಾನ ಮಾರ್ಗ** (Great circle route) ಆರ್ಕ್ಟಿಕ್ ಮೇಲಿಂದ ಹೋಗುವುದನ್ನು ಗ್ಲೋಬ್‌ನಲ್ಲಿ ಸ್ಪಷ್ಟವಾಗಿ ತೋರಿಸಬಹುದು (flat map ನಲ್ಲಿ ತಪ್ಪು ಭಾವನೆ ಸಾಧ್ಯ).

#### Globe vs Map (High-yield Comparison)

| ಅಂಶ | ಗ್ಲೋಬ್ (Globe) | ನಕ್ಷೆ (Map) |
|---|---|---|
| ಆಯಾಮ | 3D | 2D |
| ವಿಕೃತೀಕರಣ | ಇಲ್ಲ/ಅತ್ಯಲ್ಪ | ಪ್ರಕ್ಷೇಪಣದಿಂದ ಅನಿವಾರ್ಯ |
| ವಿವರ ಮಟ್ಟ | ಕಡಿಮೆ | ಹೆಚ್ಚು (especially large-scale) |
| ಉಪಯೋಗ | ಸಮಗ್ರ ಭೂಮಿಯ ಅರಿವು, ಚಲನೆಗಳು | ಸ್ಥಳೀಯ/ವಿಷಯಾಧಾರಿತ ವಿಶ್ಲೇಷಣೆ |
| ಸಾಗಣೆ | ಕಷ್ಟ | ಸುಲಭ |

### Concept 2: Map (ನಕ್ಷೆ), Cartography, Atlas

#### Definition
- ಭೂಮಿಯ ಸಂಪೂರ್ಣ ಅಥವಾ ಭಾಗದ ಆಯ್ದ ಲಕ್ಷಣಗಳನ್ನು **ನಿರ್ದಿಷ್ಟ ಮಾನಕಕ್ಕೆ (scale)** ಅನುಗುಣವಾಗಿ **ಸಮತಟ್ಟಾದ ಮೇಲ್ಮೈಯಲ್ಲಿ** ಚಿತ್ರಿಸುವ **ದ್ವಿ-ಆಯಾಮದ (2D) ನಿರೂಪಣೆ**.

#### Related Terms
- **Cartography (ನಕ್ಷಾಶಾಸ್ತ್ರ):** ನಕ್ಷೆ ರಚನೆ–ವಿನ್ಯಾಸ–ಅಧ್ಯಯನದ ವಿಜ್ಞಾನ + ಕಲೆ
- **Atlas (ಅಟ್ಲಾಸ್):** ಹಲವು ನಕ್ಷೆಗಳ ಕ್ರಮಬದ್ಧ ಸಂಗ್ರಹ

#### Key Features / Importance
- ಗ್ಲೋಬ್‌ಗಿಂತ **ಹೆಚ್ಚು ವಿವರ** ನೀಡುತ್ತದೆ
- ಹಳ್ಳಿ/ನಗರ/ರಾಜ್ಯ/ದೇಶ ಮಟ್ಟದ **ಆಳವಾದ ಅಧ್ಯಯನಕ್ಕೆ ಅನಿವಾರ್ಯ**
- ಸಾಗಿಸಲು, ಬಳಸಲು **ಸುಲಭ**
- **Thematic visualization**: ಜನಸಂಖ್ಯೆ, ಬೆಳೆ, ಸಾರಿಗೆ, ಮಳೆ, ಖನಿಜ ಇತ್ಯಾದಿ

#### Analytical Points (HSTR-level)
- ನಕ್ಷೆ = **selected representation** (ಎಲ್ಲವನ್ನೂ ತೋರಿಸುವುದಿಲ್ಲ)
- ಯಾವುದೇ ನಕ್ಷೆ ಬಳಸುವ ಮೊದಲು ಪರಿಶೀಲಿಸಬೇಕಾದವು:
  - **ಉದ್ದೇಶ/ವಿಷಯ**
  - **ಮಾನಕ (scale)**
  - **ರಚನೆಯ ದಿನಾಂಕ/ಅಪ್‌ಡೇಟ್ ಸ್ಥಿತಿ** (policy/administrative boundary changes ಗೆ ಮುಖ್ಯ)

#### Example (Exam-use)
- ನಗರ ಯೋಜನಾ ಪ್ರಾಧಿಕಾರ (ಉದಾ: ಮೆಟ್ರೋ ಮಾರ್ಗ) → **ಜನಸಂಖ್ಯಾ ಸಾಂದ್ರತೆ, ರಸ್ತೆ ಜಾಲ, ಪ್ರಮುಖ ಕೇಂದ್ರಗಳು** ತೋರಿಸುವ thematic maps ಅಗತ್ಯ; ಗ್ಲೋಬ್ ಉಪಯುಕ್ತವಲ್ಲ.

### Concept 3: Classification of Maps (ನಕ್ಷೆಗಳ ವರ್ಗೀಕರಣ)

#### आधार 1: Scale-based Classification

##### (A) Large-Scale Maps (ದೊಡ್ಡ ಪ್ರಮಾಣ)
- **ಚಿಕ್ಕ ಪ್ರದೇಶ** + **ಹೆಚ್ಚು ವಿವರ**
- ಸಾಮಾನ್ಯ ಮಾನಕ: **1:25,000 ಅಥವಾ ಅದಕ್ಕಿಂತ ದೊಡ್ಡದು** (denominator smaller → scale larger)
- Examples:
  - **Topographical maps (ಸ್ಥಳ ಸ್ವರೂಪ ನಕ್ಷೆ)**
  - **Cadastral maps (ಕಂದಾಯ ನಕ್ಷೆ)** – ಆಸ್ತಿ/ಭೂಮಿಯ ವಿವರ

##### (B) Small-Scale Maps (ಚಿಕ್ಕ ಪ್ರಮಾಣ)
- **ದೊಡ್ಡ ಪ್ರದೇಶ** + **ಕಡಿಮೆ ವಿವರ**
- ಸಾಮಾನ್ಯ ಮಾನಕ: **1:2,50,000 ಅಥವಾ ಅದಕ್ಕಿಂತ ಚಿಕ್ಕದು**
- Examples:
  - **Wall maps (ಗೋಡೆ ನಕ್ಷೆ)**
  - **Atlas maps (ಅಟ್ಲಾಸ್ ನಕ್ಷೆ)**

#### आधार 2: Purpose/Thematic-based Classification

| ಪ್ರಕಾರ | ಏನು ತೋರಿಸುತ್ತದೆ? | ಉದಾಹರಣೆ |
|---|---|---|
| Physical (ಭೌತಿಕ/ಸ್ವಾಭಾವಿಕ) | ಪರ್ವತ, ನದಿ, ಮೈದಾನ, ಪ್ರಸ್ಥಭೂಮಿ | ಕರ್ನಾಟಕ ಭೌತಿಕ ನಕ್ಷೆ |
| Political (ರಾಜಕೀಯ) | ಆಡಳಿತಾತ್ಮಕ ಗಡಿಗಳು, ರಾಜಧಾನಿಗಳು | ಕರ್ನಾಟಕ ಜಿಲ್ಲೆಗಳ ನಕ್ಷೆ |
| Thematic/Distribution (ವಿಷಯಾಧಾರಿತ/ಹಂಚಿಕೆ) | ಒಂದು ವಿಷಯದ ಹಂಚಿಕೆ (ಮಳೆ/ಜನಸಂಖ್ಯೆ/ಬೆಳೆ/ಖನಿಜ) | ಭಾರತದ ಮಳೆ ಹಂಚಿಕೆ ನಕ್ಷೆ |

#### High-yield Analytical Point (MCQ trap)
- **Scale ↔ Area inverse relationship (ವಿಲೋಮಾನುಪಾತ):**
  - **Large scale → small area → more details**
  - **Small scale → large area → less details**
- “ದೊಡ್ಡ/ಚಿಕ್ಕ” ಪದಗಳು **ಪ್ರದೇಶಕ್ಕೆ ಅಲ್ಲ**, **ಮಾನಕದ ಅನುಪಾತಕ್ಕೆ** ಸಂಬಂಧಿಸಿದವು.

### Concept 4: Elements of a Map (ನಕ್ಷೆಯ ಮೂಲಾಂಶಗಳು)

#### Definition
- ನಕ್ಷೆಯನ್ನು ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಓದಲು/ಅರ್ಥೈಸಲು ಅಗತ್ಯವಾದ **5 ಮೂಲಭೂತ ಅಂಶಗಳು**.  
- ಇವು “ನಕ್ಷೆಯ ಭಾಷೆಯ ವ್ಯಾಕರಣ”ದಂತೆ.

#### Five Essential Elements (Very Important)
1. **Title (ಶೀರ್ಷಿಕೆ)**
   - ನಕ್ಷೆಯ ವಿಷಯ/ಪ್ರದೇಶ ಸ್ಪಷ್ಟಪಡಿಸುತ್ತದೆ  
   - ಉದಾ: *“ಕರ್ನಾಟಕ – ನೈಸರ್ಗಿಕ ಸಸ್ಯವರ್ಗ”*
2. **Scale (ಮಾನಕ)**
   - ನಕ್ಷೆಯ ಅಂತರ : ಭೂಮಿಯ ನೈಜ ಅಂತರ = ಗಣಿತೀಯ ಅನುಪಾತ  
   - ರೂಪಗಳು:
     - Statement scale
     - Graphical scale
     - **R.F. (Representative Fraction)**
3. **Direction (ದಿಕ್ಕು)**
   - ಸಾಮಾನ್ಯವಾಗಿ **North Arrow (N/ಉ)** ಮೂಲಕ
4. **Grid (ಅಕ್ಷಾಂಶ–ರೇಖಾಂಶ ಜಾಲ)**
   - absolute location ನಿರ್ಧಾರಕ್ಕೆ
5. **Legend/Key (ಸೂಚಿ/ಕೈಪಿಡಿ)**
   - ಬಣ್ಣ/ಚಿಹ್ನೆ/ಸಂಕೇತಗಳ ಅರ್ಥ ವಿವರಣೆ

#### Analytical Link
- Title → “ಏನು/ಯಾವ ಪ್ರದೇಶ?”  
- Scale → “ಎಷ್ಟು ದೂರ?”  
- Direction → “ಯಾವ ದಿಕ್ಕು?”  
- Grid → “ನಿಖರ ಸ್ಥಾನ?”  
- Legend → “ಯಾವ ಲಕ್ಷಣ?”  
→ ಇವುಗಳ ಒಟ್ಟುಗೂಡಿಕೆ = **Map Reading + Map Interpretation** ಗೆ ಅಡಿಪಾಯ.

### Concept 5: Conventional Signs, Symbols & Colours (ಸಾಂಪ್ರದಾಯಿಕ ಚಿಹ್ನೆಗಳು–ಸಂಕೇತಗಳು)

#### Definition
- ನಕ್ಷೆಯಲ್ಲಿ ಭೌತಿಕ + ಸಾಂಸ್ಕೃತಿಕ ಲಕ್ಷಣಗಳನ್ನು ತೋರಿಸಲು ಬಳಸುವ **ಪ್ರಮಾಣಿತ ಚಿತ್ರಾತ್ಮಕ ನಿರೂಪಣೆಗಳು**.
- ಅಂತರರಾಷ್ಟ್ರೀಯ ಒಪ್ಪಂದದಿಂದ ಪ್ರಮಾಣೀಕರಣ → ಭಾಷಾ ಗಡಿ ಮೀರಿ ಅರ್ಥವಾಗುತ್ತದೆ.

#### Standard Colour Codes (High-yield)
- **ನೀಲಿ:** ಜಲಮೂಲಗಳು (ಸಾಗರ/ಸಮುದ್ರ/ನದಿ/ಸರೋವರ)
- **ಹಸಿರು:** ಮೈದಾನ/ತಗ್ಗು ಪ್ರದೇಶ
- **ಹಳದಿ:** ಪ್ರಸ್ಥಭೂಮಿ/ಮೈದಾನಕ್ಕಿಂತ ಎತ್ತರದ ಭೂಮಿ
- **ಕಂದು:** ಬೆಟ್ಟ/ಪರ್ವತ (ಎತ್ತರ ಹೆಚ್ಚಾದಂತೆ ಗಾಢ)
- **ಬಿಳಿ:** ಹಿಮಾವೃತ ಪ್ರದೇಶ

#### Common Conventional Symbols (Examples)
- **Boundaries (ಗಡಿಗಳು):**
  - ಅಂತರರಾಷ್ಟ್ರೀಯ → ದಪ್ಪ ಚುಕ್ಕಿ-ಗೆರೆ
  - ರಾಜ್ಯ → ಸಣ್ಣ ಚುಕ್ಕಿ-ಗೆರೆ
  - ಜಿಲ್ಲೆ → ಸರಳ ಗೆರೆ
- **Transport:**
  - ಡಾಮರು ರಸ್ತೆ → ಎರಡು ಕೆಂಪು ಸಮಾಂತರ ರೇಖೆಗಳು
  - ರೈಲು ಮಾರ್ಗ → ಕಪ್ಪು ಗೆರೆ + ಅಡ್ಡಪಟ್ಟಿಗಳು
- **Buildings/Institutions:** ದೇವಸ್ಥಾನ/ಚರ್ಚ್/ಮಸೀದಿ/ಕೋಟೆ/PO ಇತ್ಯಾದಿಗೆ ನಿರ್ದಿಷ್ಟ ಚಿಹ್ನೆಗಳು
- **Natural:** ನದಿ, ಬಾವಿ, ಬುಗ್ಗೆ, ಮರಗಳು, ಹುಲ್ಲುಗಾವಲು

#### Feature–Symbol (Revision Table)
| Feature (ಲಕ್ಷಣ) | Conventional symbol (ಪರೀಕ್ಷಾ ದೃಷ್ಟಿ) |
|---|---|
| River/Water body | Blue line/patch |
| Metalled road | Two red parallel lines |
| Railway line | Black line with cross bars |
| Boundary types | Different dotted/solid line styles |
| Temple/Church/Mosque | Standard pictorial icons |
| Well/Spring | Standard small circular/mark symbols |

#### Analytical Point
- **Legend (ಸೂಚಿ)** ಓದುವ ಕೌಶಲ್ಯ + symbols ಜ್ಞಾನ = ನಕ್ಷೆಯನ್ನು “ಜಾಗತಿಕ ಸಂವಹನ ಮಾಧ್ಯಮ”ವಾಗಿಸುತ್ತದೆ.

## 4. Important Places (State-wise if relevant)

### Karnataka Focus (Exam Utility)
- **ಕರ್ನಾಟಕ ರಾಜಕೀಯ ನಕ್ಷೆ**:
  - ಜಿಲ್ಲೆಗಳು, ಜಿಲ್ಲಾ ಕೇಂದ್ರಗಳು, ಗಡಿಗಳು (ರಾಜ್ಯ/ಜಿಲ್ಲೆ) ಗುರುತಿಸುವುದು
  - **Graphical scale ಬಳಸಿ ಎರಡು ಜಿಲ್ಲಾ ಕೇಂದ್ರಗಳ ನಡುವಿನ ಅಂತರ** ಅಳೆಯುವ ಅಭ್ಯಾಸ (ಉದಾ: ಕಲಬುರಗಿ–ಬಾಗಲಕೋಟೆ ಮಾದರಿ)
- **Map pointing importance**
  - GPST/HSTR ನಲ್ಲಿ “ಜಿಲ್ಲಾ ಕೇಂದ್ರ ಗುರುತು”, “ಗಡಿ ರಾಜ್ಯಗಳು”, “ರಾಜಧಾನಿ” ಪ್ರಶ್ನೆಗಳು ಬರುತ್ತವೆ.
  - Political map + direction sense (N, E, W, S) → location-based MCQs.

## 5. Analytical Understanding (Very Important)

### Cause → Effect Links
- **ಭೂಮಿ ಗೋಳಾಕಾರ (Geoid) + ಸಮತಟ್ಟಾದ ನಕ್ಷೆ ಅಗತ್ಯ**  
  → ನಕ್ಷೆಗಳಲ್ಲಿ **ಪ್ರಕ್ಷೇಪಣ ಸಮಸ್ಯೆ** (distortion) ಅನಿವಾರ್ಯ  
  → ಗ್ಲೋಬ್ distortion-free reference tool ಆಗಿ ಉಳಿಯುತ್ತದೆ.
- **Scale ಆಯ್ಕೆ**  
  → ವಿವರ ಮಟ್ಟ ನಿರ್ಧಾರ  
  → planning/administration (ನಗರ ಯೋಜನೆ, ಕಂದಾಯ) ಗೆ large-scale ಅಗತ್ಯ.

### Technology → Economy → Politics (Applied angle)
- **Surveying + Cartography + GIS/Remote Sensing**  
  → ನಿಖರ ನಕ್ಷೆಗಳು  
  → ನಗರ ಮೂಲಸೌಕರ್ಯ (ಮೆಟ್ರೋ/ರಸ್ತೆ), ವಿಪತ್ತು ನಿರ್ವಹಣೆ, ಕೃಷಿ ಯೋಜನೆ  
  → ಆಡಳಿತಾತ್ಮಕ ನಿರ್ಧಾರಗಳು (boundaries, resource allocation) ಹೆಚ್ಚು ಡೇಟಾ-ಆಧಾರಿತ.

### Social Transformation Points
- ನಕ್ಷೆಗಳು:
  - ಸಾರ್ವಜನಿಕ ಸೇವೆಗಳ ಯೋಜನೆ (transport, health, education) ಗೆ spatial evidence
  - ಜನಸಂಖ್ಯಾ ಸಾಂದ್ರತೆ/ಸಂಪನ್ಮೂಲ ಹಂಚಿಕೆ ವಿಶ್ಲೇಷಣೆ ಮೂಲಕ **ನೀತಿ ರೂಪಣೆ**ಗೆ ನೆರವು

### Evolution Pattern (Skill-based)
- **Map Reading → Map Interpretation → Spatial decision-making**
  - Reading: symbols/scale/direction ಗುರುತಿಸುವುದು
  - Interpretation: ಕಾರಣ-ಪರಿಣಾಮ, ಹಂಚಿಕೆ ಮಾದರಿ, ಪ್ರಾದೇಶಿಕ ವ್ಯತ್ಯಾಸ ವಿಶ್ಲೇಷಣೆ

## 6. Previous Exam Focus Areas

### 1–2 Mark Facts (Direct)
- ಗ್ಲೋಬ್ = 3D, distortion-free; ನಕ್ಷೆ = 2D, scale-based
- Cartography, Atlas ವ್ಯಾಖ್ಯಾನ
- ನಕ್ಷೆಯ 5 ಮೂಲಾಂಶಗಳು: Title, Scale, Direction, Grid, Legend
- ಬಣ್ಣ ಸಂಕೇತಗಳು: Blue-water, Brown-mountains, Green-plains, White-snow

### 5 Mark Descriptive Areas
- **Globe vs Map**: ಉಪಯೋಗಗಳು + ಮಿತಿಗಳು + ಯಾವ ಸಂದರ್ಭದಲ್ಲಿ ಯಾವುದು ಸೂಕ್ತ?
- **Scale-based classification**: large vs small scale + ಉದಾಹರಣೆಗಳು
- **Map elements**: ಪ್ರತಿ ಅಂಶದ ಪಾತ್ರ + map reading/interpretation ಗೆ ಸಂಬಂಧ

### MCQ Traps (Common)
- “Large scale” ಎಂದರೆ **ದೊಡ್ಡ ಪ್ರದೇಶ** ಎಂದು ತಪ್ಪಾಗಿ ಆಯ್ಕೆ ಮಾಡುವುದು (ವಾಸ್ತವ: **ಚಿಕ್ಕ ಪ್ರದೇಶ + ಹೆಚ್ಚು ವಿವರ**)
- Legend ಮತ್ತು Title ಗೊಂದಲ (Legend = symbols explanation; Title = theme/area)
- Colour codes: green vs yellow (green = lowland/plains; yellow = higher land/plateau)

### Frequently Confused Points
- **Scale “ದೊಡ್ಡ/ಚಿಕ್ಕ”** = R.F. denominator ಆಧಾರಿತ (1:25,000 is larger than 1:2,50,000)
- **Political map vs Physical map**: boundaries/capitals vs natural relief
- **Conventional symbols** vs **Legend**:
  - Symbols = standardized signs
  - Legend = ಆ ನಕ್ಷೆಯಲ್ಲಿ ಬಳಸಿದ symbols ಗಳ ಪಟ್ಟಿ/ಅರ್ಥ
